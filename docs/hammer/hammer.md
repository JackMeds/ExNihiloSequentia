Hammers
=======

Block Breakdown
---------------
The following is what a block will break down into.
| Original    | Output             |
|-------------|--------------------|
| Stone       | Cobblestone        |
| Cobblestone | Gravel             |
| Andesite    | Crushed Andesite   |
| Granite     | Crushed Granite    |
| Diorite     | Crushed Diorite    |
| Gravel      | Sand               |
| Sand        | Dust               |
| End Stone   | Crushed End Stone  |
| Netherrack  | Crushed Netherrack |
